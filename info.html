Setup Information

Create Service Account
Service Account Required: You need a Google Cloud service account to access your Google Sheets. Follow these steps to create one:
  1. Go to Google Cloud Console
    Open Google Cloud Console
  2. Create a new project or select an existing one
    If you don't have a project, create one with any name
  3. Enable the Google Sheets API:
    • Go to APIs & Services → Library
    • Search for "Google Sheets API" and enable it
  4. Create Service Account credentials:
    • Go to APIs & Services → Credentials
    • Click "Create Credentials" → "Service Account"
    • Fill in service account details and click "Create"
    • Skip optional steps and click "Done"
  5. Download the JSON key file:
    • Click on your newly created service account
    • Go to the "Keys" tab
    • Click "Add Key" → "Create new key"
    • Choose "JSON" format and download the file
    • Upload this file in the Google Sheets configuration section
Important Notes:
  • Keep your service account JSON file secure
  • Share your Google Sheet with the service account email
  • The service account email looks like: name@project.iam.gserviceaccount.com
Quick Tips:
  • You only need to create the service account once
  • The same service account can be used for multiple sheets
  • Make sure to enable billing if required by Google Cloud
GitLab Personal Access Token
GitLab Token Required: You need a personal access token to access your GitLab projects. Here's how to create one:
  1. Go to your GitLab profile settings
    Click on your profile picture → Settings → Access Tokens
  2. Create a new personal access token
    • Give it a descriptive name (e.g., "Sheet Ninja Sync")
    • Set an expiration date (recommended)
    • Select the "api" scope
    • Click "Create personal access token"
  3. Copy and save the token
    ⚠️ Important: Copy the token immediately as you won't be able to see it again!
