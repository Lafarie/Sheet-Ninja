Date Range Filter (Optional)
If enabled, only tasks within the selected date range will be processed. If disabled, all tasks without GIT_ID will be processed.
Task Closing Options
Tick: Tasks with status "Done", "Completed", "Finished", or "Closed" will be automatically closed in GitLab immediately.
Untick: Tasks will only be created/updated, but never automatically closed.
Workflow Options:
• Immediate Closure: Close tasks right away when marked as done
• Manual Review: Create/update tasks but close them manually later
