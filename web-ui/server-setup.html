🚀 Local Server Setup

To run the Google Sheets Manager locally and avoid CORS issues, you need to serve the files through a local server. Choose one of the methods below:

Method 1: Python Server (Recommended)

If you have Python installed:

Python 3:

cd /Users/farhadlafarie/projects/scripts/web-ui
python -m http.server 8000

Python 2:

cd /Users/farhadlafarie/projects/scripts/web-ui
python -m SimpleHTTPServer 8000

Then open: http://localhost:8000

Method 2: Node.js Server

If you have Node.js installed, install http-server:

npm install -g http-server
cd /Users/farhadlafarie/projects/scripts/web-ui
http-server -p 8000

Then open: http://localhost:8000

Method 3: PHP Server

If you have PHP installed:

cd /Users/farhadlafarie/projects/scripts/web-ui
php -S localhost:8000

Then open: http://localhost:8000

Method 4: Live Server Extension (VS Code)

If you're using VS Code:

  1. Install the "Live Server" extension
  2. Right-click on index.html
  3. Select "Open with Live Server"
⚠️ Important: Due to browser security policies, the Google Sheets API integration requires a proper server setup. Opening the HTML file directly in a browser (file://) will not work for API requests.
