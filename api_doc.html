API Documentation

This tool uses the GitLab API v4 and Google Sheets API to synchronize data between platforms.
GitLab API Endpoints
  • GET /projects/{id}/issues - List project issues
  • POST /projects/{id}/issues - Create new issue
  • PUT /projects/{id}/issues/{iid} - Update issue
  • GET /projects/{id}/labels - List project labels
  • GET /projects/{id}/milestones - List project milestones
Google Sheets API
  • spreadsheets.get - Read spreadsheet data
  • spreadsheets.values.update - Update cell values
  • spreadsheets.values.batchUpdate - Batch update values
  • spreadsheets.batchUpdate - Batch operations
