📋 How to Setup GitLab Integration:

  1. Personal Access Token:
    • Go to GitLab → Settings → Access Tokens
    • Create token with "api" scope
    • Copy and save the token securely
  2. Project Access:
    • Must have Developer role or higher
    • Issues feature must be enabled
  3. Data Format:
    • Expected headers: Date, GIT ID, Project Name, Task Description, Status, etc.
    • Task Description becomes issue title
    • Other fields become issue description
ℹ️ Help & Information

🔧 How to Get Google Sheets API Credentials:

  1. Go to Google Cloud Console
  2. Create a new project or select existing one
  3. Enable the Google Sheets API
  4. Create credentials (Service Account)
  5. Download the JSON file

📋 Supported Features:

  • ✅ Multiple sheet support
  • ✅ Headers and full data export
  • ✅ JSON format download
  • ✅ Real-time data preview

  • ✅ Mobile-responsive design
💡 Tip: Make sure to share your Google Sheet with the service account email address found in your credentials JSON file.
