Sync Operations

Perform synchronization between GitLab and Google Sheets

Sheets → GitLab

Sync changes from Google Sheets to GitLab:

  • Create new GitLab issues for tasks without GIT ID
  • Update existing issues based on status changes
  • Close completed issues
  • Add time tracking information
GitLab → Sheets

Sync changes from GitLab to Google Sheets:

  • Update issue statuses in the sheet
  • Sync time tracking information
  • Update due dates and milestones
  • Reflect GitLab changes in the sheet
Complete Sync

Perform bidirectional synchronization:

  • GitLab → Sheets sync first
  • Then Sheets → GitLab sync
  • Ensures both systems are up to date
  • Recommended for regular synchronization

Pre-sync Checklist
Before Starting Sync:
  • ✓ Configuration is complete
  • ✓ Connections are tested
  • ✓ Column mappings are correct
  • ✓ No one else is editing the sheet
Best Practices:
  • Run syncs during low-activity periods
  • Review changes before syncing
  • Keep backups of important data
  • Monitor sync output for errors
Troubleshooting:
  • Check network connectivity
  • Verify API tokens are valid
  • Ensure proper permissions
  • Review error messages in output
